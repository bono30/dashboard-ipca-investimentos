Dashboard Unificado: IPCA + Simulador de Investimentos

Este arquivo preserva os dois dashboards originais dentro de uma única interface. Os cálculos, gráficos e interações de cada módulo permanecem independentes para evitar conflitos de JavaScript e manter o funcionamento real.

Use as abas acima para alternar entre os módulos. Este arquivo foi montado para manter a lógica original de cada dashboard intacta. Caso o navegador bloqueie alguma API externa, abra o arquivo com internet ativa e permita o carregamento dos recursos.
